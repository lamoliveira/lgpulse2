# Como funciona?

Use o LG - Pulse para potencializar a análise em seus aplicativos, da análise à visualização,
geração de insights e além. Este artigo se concentra nos detalhes de alto nível da
integração para ajudar você a começar.

Casos de uso comuns para adicionar Dashboards ao seu aplicativo (também conhecido como *análise incorporada* ou embedded) incluem:

- painéis e gráficos como componentes de suas aplicações 
- Relatórios automatizados com dados atualizados e uma experiência de usuário de qualidade

![embedded analytics](/img/superlogica/mensal.png)

Além de ter acesso aos dashboards já customizados por nossa equipe de especialistas, você ainda poderá customizar seus próprios dashboards

## 1. Analise seus dados 

To add analytics to your app, you start with [LG Desktop](https://www.tableau.com/products/desktop) to
connect to your data sources. This application empowers you to explore data like you've never experienced before!
Analyzing data visually is at the core of Tableau's claim to fame.

Para adicionar análises ao seu aplicativo, você começa com o LG Analytics para
conectar-se às suas fontes de dados. Este aplicativo permite que você explore dados como nunca experimentou antes!
A análise visual de dados é o que vai permitir seu time criar a cultura de dados para se tornar uma área data driven.

## 2. Publique seu trabalho

Para compartilhar seu trabalho com outros, você precisa de um ambiente para hospedar esse novo conteúdo e protegê-lo para que ele seja acessível apenas
aos visualizadores com quem você pretende compartilhar esses insights. Você tem a opção de publicar com um novo nome para que outras pessoas também tenham acesso na sua organização.
