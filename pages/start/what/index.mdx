# LG Pulse

LG Pulse é uma plataforma, uma comunidade e um ecossistema. A tecnologia que o impulsiona permite que as pessoas vejam e entendam dados
dentro de um ambiente seguro e escalável.

![Exemplo despesas](/img/superlogica/despesas.png)

Analisar dados requer ferramentas que podem se conectar a praticamente qualquer das suas fontes de dados e são úteis para visualização,
trabalhando com modelos estatísticos e tudo isso é auxiliado pela inteligência artificial.

Análises bem-sucedidas precisam ser comunicadas a outros para causar impacto. Este é um ponto forte adicional do LG Pulse,
pois seus produtos poderosamente criativos permitem que indivíduos compartilhem insights com segurança em um ambiente protegido.
